簡單 4 步，馬上開始

BTalk 使用 Groq 提供極速的 AI 語音辨識，設定一次就能永久免費使用。

1

下載並安裝應用程式

點擊上方的下載按鈕取得最新的 .dmg 檔案，打開後將 BTalk 拖曳到你的「應用程式」資料夾中。

2

授權麥克風與輔助使用權限

第一次開啟 BTalk 時，系統會要求「麥克風」與「輔助使用」權限。
請依畫面指示前往 macOS 的「系統設定」，將 BTalk 打勾允許，這樣才能順利收音並幫你自動打字喔！

3

獲取免費的 Groq API Key

你需要一組免費的 API 密鑰來啟用 AI 語音辨識：

  • 前往 Groq Console 並註冊 / 登入帳號。
  • 點擊畫面右上角的 "Create API Key" 按鈕。
  • 輸入一個好記的名稱（例如 BTalk），然後複製生成的那串密碼。
4

貼上密鑰，開始變態輸出！

打開 BTalk 應用程式，在設定畫面中貼上剛剛複製的 Groq API Key。
設定好你的快捷鍵，現在你可以隨時隨地用語音輸入文字了！
